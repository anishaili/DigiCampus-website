Introduction

In this intensive 10-day course, you will dive into the world of front-end development starting with the solid foundations of HTML and CSS. You will learn to structure your web pages with HTML and style them with CSS to create aesthetic and user-friendly interfaces.

Modules

Understanding the Web

Understanding the web involves grasping the fundamental concepts and technologies that power the World Wide Web. This includes understanding how web browsers and web servers communicate using HTTP, the role of HTML, CSS, and JavaScript in creating web pages, and client-server architecture concepts. This also involves understanding concepts like DNS (Domain Name System), URLs (Uniform Resource Locators), and web standards. Overall, understanding the web means understanding how information is accessed, presented, and interacted with on the internet.

Designing in the Browser

Designing in the browser refers to the ability to create graphics, animations, and other visual elements directly in a web browser using technologies like HTML5 canvas, SVG (Scalable Vector Graphics), and WebGL (Web Graphics Library). These technologies allow developers to design shapes, lines, text, and images dynamically, enabling the creation of interactive web applications visually without relying on external plugins or software. Browser design also involves creating visually appealing user interfaces and experiences using data visualizations and other interactive elements directly in the web browser.
